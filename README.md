# 🔐 Secure Login System (Flask)

## 📌 Features
- User Signup & Login
- Password Hashing (Security)
- Session Management
- Dashboard Page
- Clean UI with CSS

## 🛠 Tech Stack
- Python (Flask)
- SQLite
- HTML, CSS

## 🚀 How to Run
1. Install dependencies:
   pip install flask flask-bcrypt

2. Run:
   python app.py

3. Open browser:
   http://127.0.0.1:5000/

## 📷 Output
- Login Page
- Signup Page
- Dashboard
